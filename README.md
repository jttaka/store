# store
A small fun script for easy access to internal drives. Modify this script to your liking.

Requirements:
dunst

Install dunst:
Arch:
```bash
sudo pacman -S dunst
```
Debian:
```bash
sudo apt install dunst
```
MMake sure to change [YOURDRIVE] and [YOUR-USER-NAME]
to your drives name and your user name in the store file
```bash
DEV="/dev/[YOURDRIVE]"
MNT="/home/[YOUR-USER-NAME]storage"
```
